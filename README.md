# CyberGuardianBot

## Description
This is a C# console chatbot designed to educate users about cybersecurity awareness. It provides interactive responses on topics such as password safety, phishing, and safe browsing.

## Features
- Voice greeting using WAV file
- ASCII art logo
- User name personalisation
- Input validation
- Cybersecurity tips
- Enhanced console UI with colours and typing effect

## How to Run
1. Open the project in Visual Studio
2. Ensure greeting.wav is in the project directory
3. Run the program
